Requisitos Não Funcionais (RNF)
RNF-01 — Linguagem

O sistema deve ser desenvolvido em Python 3.

RNF-02 — Framework

A API deve ser desenvolvida utilizando FastAPI.

RNF-03 — Banco de Dados

O sistema deve utilizar SQL Server como banco de dados relacional.

RNF-04 — Padrão REST

A API deve seguir os padrões REST.

RNF-05 — Documentação

A API deve estar documentada utilizando o padrão OpenAPI (Swagger).
